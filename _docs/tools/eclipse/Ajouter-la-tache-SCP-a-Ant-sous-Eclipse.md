Aller à cette adresse pour télécharger le jar : http://www.jcraft.com/jsch/index.html 

Puis coller le jar dans `~/.ant/lib` pour linux ou `C:\Documents and Settings\<user>\.ant\lib` pour Windows

Relancer Eclipse

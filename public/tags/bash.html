Comparaisons avec diff

Quelques commandes diff pour comparer des répertoires et des fichiers : Répertoires Comparaison de répertoire en récursif sans tenir compte des espaces : diff -rqwB rep-original rep-modifié | sort > modification-rep.diff Fichiers Comparaison des fichiers sans tenir compte des espaces et au format universel (patch): diff -wBu fichier-original.txt fichier-modifié.txt > fichier-diff. »

Connexion VPN Rackspace depuis Linux

Sous Linux il n’existe pas de client VPN Cisco officiel mais Rackspace préconise l’utilisation de VPNC qui fonctionne très bien. Installation du package Sous Debian & Co c’est simple : $ sudo apt-get install vpnc Configuration Commencez par créer un fichier dans /etc/vpnc $ sudo vi /etc/vpnc/rackspace.conf Ajoutez y les lignes suivante : IPSec gateway 184.106.120.83 IPSec ID CameleonAdmins IPSec secret {password group ? voir avec RSE} Xauth username fcombes@cameleon-software. »

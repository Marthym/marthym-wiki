Requête Oracle avec Variables Liées

C’est un info qui date mais c’est toujours juste. C’est quoi des Bind Variable déjà ? Dans le processus d’exécution d’une requête, Oracle effectue plusieurs étapes. Parmi ces étapes, on retrouve le parsing. Durant cette étape, oracle décortique l’ordre SQL et choisit quel est le plan d’exécution le plus court pour récupérer les données ramenées par la requête. Si une requête est exécuté 20 fois de suite, Oracle fait le parsing pour la première requête seulement. »

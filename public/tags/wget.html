WGET avec un proxy

WGET est un programme en ligne de commande qui permet de télécharger des fichiers depuis le web. Pour une utilisation en entreprise, il se peut qu’un proxy filtre les accès au web. ## Utiliser WGET avec un proxy simple Créer un fichier .wgetrc (n’oubliez pas le point devant le fichier) a la racine de votre répertoire personnel avec le contenu suivant : http_proxy = http://votre_proxy:port_proxy/ use_proxy = on wait = 15 Vérifiez que ca fonctionne en rapatriant un fichier test : »
